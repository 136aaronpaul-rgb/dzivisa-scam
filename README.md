# dzivisa-scam
A tool to help detect and prevent scams in Zimbabwe.

## Features
- SMS/WhatsApp scam pattern detection
- Report and verify suspicious numbers
- Community blacklist database

## How to run
```bash
git clone https://github.com/136aaronpaul-rgb/dzivisa-scam.git
cd dzivisa-scam
pip install -r requirements.txt
python main.py
